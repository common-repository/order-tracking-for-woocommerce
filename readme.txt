== Description ==

= Introduction =
Easily track orders placed on your e-commerce website. Simply use your order number and email to find the status of your order.

== Installation ==

1. Upload the plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Thats it! you can now configure the plugin.

== Screenshots ==

1. Add tracking details to order
